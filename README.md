## 📁 Project Structure

```bash
/pages         # Website routes (Home, Blog)
/components    # Reusable UI components (Navbar, Footer, etc.)
/content       # Blog post data (JSON or Markdown)
/public        # Static files (images, icons, etc.)
/styles        # Global CSS (Tailwind config, if needed)
```
